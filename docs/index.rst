Welcome to RevelioNN's documentation!
=====================================

   .. |RevelioNN logo| image:: images/logo.png
      :width: 100%

RevelioNN is an open-source library of post-hoc algorithms for explaining predictions of deep convolutional neural
networks of binary classification using ontologies. The algorithms are based on the construction of mapping networks
linking the internal representations of a convolutional neural network with ontology concepts.
